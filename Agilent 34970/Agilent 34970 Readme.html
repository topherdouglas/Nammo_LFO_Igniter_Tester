Agilent 34970 Instrument Driver Readme

To rate this driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Agilent Technologies
Supported Language(s): LabVIEW 8.2
Supported Model(s): Agilent 34970, Agilent 34972A
Interface(s): 34970 - GPIB, Serial. 34972 - USB, Ethernet, LXI
Firmware Revision Tested: 1.11-1.10-02-01

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 3.1
Original Release Date: February 1998
Current Revision Date: March 11, 2010

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later

For more details, refer to the LabVIEW Help under Help » VI, Function, & How-To Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please contact technical support. If you supply an invalid channel for configuring current, the visa session will freeze up.
Do not use the EZ voltage.vi after the alarm configurations. The EZ voltage.vi will erase the alarm configurations.
